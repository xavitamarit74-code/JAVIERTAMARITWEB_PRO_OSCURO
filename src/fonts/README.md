# Fuentes de Montserrat Alternates

Coloca aquí los archivos de fuentes `.woff2`:

- `montserrat-alternates-v17-latin-200.woff2`
- `montserrat-alternates-v17-latin-300.woff2`
- `montserrat-alternates-v17-latin-regular.woff2`
- `montserrat-alternates-v17-latin-500.woff2`
- `montserrat-alternates-v17-latin-600.woff2`
- `montserrat-alternates-v17-latin-700.woff2`
- `material-icons.woff2`

Puedes descargar las fuentes desde:

- Montserrat Alternates: <https://google-webfonts-helper.herokuapp.com/fonts/montserrat-alternates>
- Material Icons: <https://google.github.io/material-design-icons/>

Gulp copiará automáticamente estos archivos a `build/fonts/` cuando ejecutes `npm run dev` o `npm run build`.
