# Archivo de ejemplo Markdown

Este es un archivo Markdown de ejemplo para la carpeta `txt`.

## Características

- La carpeta `txt` está configurada en Gulp
- Se copian automáticamente archivos `.txt` y `.md`
- Aparecerán en `dist/txt/` después del build

## Uso

Simplemente añade tus archivos `.txt` o `.md` a esta carpeta y se incluirán automáticamente en el build del proyecto.

---

*Configurado por Gulp en `gulpfile.js`*
